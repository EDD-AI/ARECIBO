# ARECIBO — Salvage Protocol

> Jeu navigateur coop d'extraction spatiale post-apocalyptique.

## Concept

Un vaisseau-épave dans l'espace. Des joueurs qui sortent récupérer des fragments d'une boîte mystérieuse envoyée par l'humanité. Chaque session améliore le vaisseau. Des transmissions chiffrées créent une tension sociale entre les joueurs.

## Stack

- HTML / CSS / JS vanilla (front)
- Esthétique SF vintage retrofuturiste (Alien 1979, 2001)
- Fonts : Power Grotesk (logo) · Degular (titres) · Pixelify Sans (UI pixel)
- Palette : #F7FFC3 (cream) · #FA5A1F (orange) · #000 (bg)

## Flow

`Boot terminal` → `Menu` → `Solo (création perso)` ou `Multi (lobby)` → `Session extraction`
